### Solar Watch Site

---

This project is a website where the users are able to check the sunrise/sunset data for specific cities.


### Implemented Features

---

- Landing page with navbar
- Registration feature
- Authentication feature
- Log out feature
- Protect routes with authentication
- Search for city's sunset/sunrise data
- store data to database upon first API request to minimize requests


### Technologies Used

---

- Spring Boot
- Spring Data JPA
- Spring Security
- ReactJS
- PostgreSQL
- Docker
- Bootstrap
- [Sunrise Sunset API](https://sunrise-sunset.org/api)
- [Geocode API](https://openweathermap.org/api/geocoding-api)
